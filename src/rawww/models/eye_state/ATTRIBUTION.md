# Модель определения открытых/закрытых глаз

- **Файл:** `mobilenetv2_eyes.onnx`
- **Модель:** Open-Closed Eye Classification (MobileNetV2)
- **Автор:** Michał Młodawski
- **Источник:** https://huggingface.co/MichalMlodawski/open-closed-eye-classification-mobilev2
- **Лицензия:** Creative Commons Attribution-NonCommercial-NoDerivatives 4.0
  International (CC BY-NC-ND 4.0) — https://creativecommons.org/licenses/by-nc-nd/4.0/
- **Классы:** `0` — глаза закрыты, `1` — глаза открыты.
- **Вход:** RGB 224×224, нормализация mean=std=0.5.

Модель поставляется без изменений (NoDerivatives) и используется только для
инференса. Разрешено некоммерческое использование (NonCommercial); при
распространении сохраняйте это указание авторства.
